Activity Specs:

1. TEXTSIZE_1 - Increases the app-wide text size by one step each time it is tapped. Allows the user to make text larger for better readability.
2. TEXTSIZE_2 - Decreases the app-wide text size by one step each time it is tapped. Allows the user to make text smaller to fit more content on screen.
3. REMINDER_1 - nables or disables the Daily Verse push notification reminder. When checked (ON), the app sends a daily Bible verse notification to the user. Default state is ON              (checked).
4. AUDIO_1 - Controls the playback volume for the Audio Bible. The user drags the slider left to decrease or right to increase volume. Range: 0.0 (mute) to 1.0 (maximum).
5. AUDIO_5 - Tappable row labeled 'Audio Bible Voice' with a chevron (>) icon. Navigates the user to a sub-screen where they can select their preferred Bible narrator or voice.
