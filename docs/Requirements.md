User Stories:
  - As a user, I want to read the Bible by book, chapter, and verse.
  - As a user, I want to scroll smoothly or swipe between chapters so that reading feels easy on mobile.
  - As a user, I want to see the scripture of the day.
  - As a user, I want to search by book name.
    
Requirements:
  - The system must allow users to view Bible books, chapters, and verses.
  - The system must allow users to navigate between chapters (next/previous).
  - The system must display text in a scrollable format.
  - The system must allow users to adjust font size.
  - The system must display the scripture of the day.
  - The system must play ambient noise when reading which can be toggled.
