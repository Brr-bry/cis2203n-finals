User Stories:
  US_01 - As a user, I want to read the Bible by book, chapter, and verse.  
  US_02 - As a user, I want to scroll smoothly in chapters so that reading feels easy.  
  US_03 - As a user, I want to see the scripture of the day.  
  US_04 - As a user, I want to search by book name.  
  US_05 - As a user, I want to change the font size of the text.
